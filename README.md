# 🏆 CobbleGyms

Sistema completo de Gyms, E4 y Campeón Pokémon para servidores **Cobblemon** en **Minecraft 1.21.1 (Fabric)**.

| 10 - Pulido | Config, i18n, testing, docs | ⏳ Pendiente |

## Instalación

1. Instala [Fabric Loader](https://fabricmc.net/) para Minecraft 1.21.1
2. Descarga [Fabric API](https://modrinth.com/mod/fabric-api)
3. Descarga [Cobblemon](https://modrinth.com/mod/cobblemon) 1.6.x para 1.21.1
4. Coloca `CobbleGyms.jar` en la carpeta `mods/`
5. Configura `config/cobblegyms.json` a tu gusto

## Requerimientos

- Minecraft 1.21.1
- Fabric Loader ≥ 0.15.11
- Fabric API
- Cobblemon ≥ 1.6.1 para 1.21.1
- Java 21+
